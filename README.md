# Sensor intelligence Lab

**Author** Fausto Orozco Coy, MSc.
**Status** Active Development (Week 1)

## Overview
This repository contains the firmware and DSP pipelines for high-fidelity signal acquisition using ESP32.

## Structure
- `/firmware`: ESP32 SOURCE CODE (c++/PlatformIO)
-`/tools`: Python scripts for data logging and signal processing.
- `/data`: Storage for captured dataset (Git ignored).
- `/docs`: Engineering logs and datasheets.

## Setup
1. Install PlatformIO extension un VS Code.
2.  Python dependencies: `pip install -r tools/requirements.txt`
